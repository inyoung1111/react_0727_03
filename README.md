# Next.js
- React를 기반으로 한 프레임워크로 SSR, CRS 모두가능
- 풀스택 프레임워크 백엔드 개발 가능
- 클라이언트 측에서만 실행되어야 하는 코드를 명시할 때 사용하는 선언
'use client';
-src/app 안에  tailwindCSS가 적용되는 폴더 및 파일들을 넣는다.
